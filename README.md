# YouCloud Desktop

Премиальный, современный настольный клиент для **SoundCloud**, построенный с использованием **Kotlin**, **Compose Multiplatform** и **Material 3**. 

Интерфейс приложения адаптирован под современные стандарты веб-дизайна и включает в себя динамическую смену состояний, сглаженные углы, гармоничные цветовые схемы и плавные анимации.

---

## Основные возможности

* 🎨 **Интерфейс в стиле Material 3**: Динамическое переключение тем оформления, плавная регулировка громкости, поддержка интерактивных микро-анимаций для управляющих кнопок.
* 🎧 **Интеграция с системой (MPRIS на Linux)**: Полная интеграция с системным микшером и панелью управления медиа (отображение обложки, названия трека, автора и управление через клавиатурные медиа-клавиши).
* 💾 **Офлайн-режим (Downloads)**: Скачивайте любимые треки на устройство для прослушивания без подключения к интернету.
* 📂 **Управление плейлистами**: Создавайте локальные плейлисты, добавляйте в них треки прямо из контекстного меню.
* ⌨️ **Удобство использования**: Быстрое управление воспроизведением (пауза/старт) с помощью клавиши `Space` (пробел автоматически игнорируется при активном вводе текста в поля поиска или настроек).
* 🔍 **Умный поиск**: Мгновенный поиск треков по SoundCloud.

---

## Требования для сборки и работы

1. **Java Development Kit (JDK) 21** или выше.
2. **VLC Media Player**: Для воспроизведения аудиофайлов используется библиотека `vlcj`. Убедитесь, что VLC установлен в вашей системе:
   * **Ubuntu/Debian**: `sudo apt install vlc`
   * **macOS**: `brew install --cask vlc`

## Инструкция по запуску и сборке

### Запуск в режиме разработки
```bash
./gradlew run
```

### Создание переносимой (portable) сборки
Собирает готовую сборку приложения со встроенным JDK в папку `build/compose/binaries/main/app/` (не требует предустановленной Java на целевом компьютере):
```bash
./gradlew createDistributable
```

### Создание установочных пакетов
* **На Linux (сборка .deb пакета):**
  ```bash
  ./gradlew packageDeb
  ```
* **На macOS (сборка .dmg пакета):**
  ```bash
  ./gradlew packageDmg
  ```

## Настройка авторизации

Для полноценного использования поиска и синхронизации лайков перейдите во вкладку настроек в приложении и укажите:
1. **OAuth-токен** вашего аккаунта SoundCloud.
2. **Client ID** (при необходимости тонкой настройки API).
